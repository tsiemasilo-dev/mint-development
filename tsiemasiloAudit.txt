  Order Email Notification System — Built from Scratch  [MarDeveloper]
  · Built the complete automated order email system using Resend:
              - order_emails table in Supabase (one row per order)
              - sendOrderConfirmationEmail() — fires immediately after payment
              - sendOrderFillEmail() — fires on CSDP/broker settlement (T+3)
              - Idempotency checks to prevent any duplicate sends
  · Split into two distinct email types: Confirmation and Settlement Fill
  · Redesigned both email templates to match the Mint brand:
              - Deep purple gradient header
              - Bordered detail cards per order field
              - Amber "Pending Settlement" / Purple "Settled" status badges
              - Purple CTA button — Track Your Order / View My Portfolio
  · Consolidated to one shared template file used by both Vercel serverless
    functions and the Express server — single source of truth
